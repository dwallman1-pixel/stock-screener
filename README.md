# stock-screener
A real-time stock screener powered by TradingView widgets
# Clone the repository
git clone https://github.com/dwallman1-pixel/stock-screener.git
cd stock-screener

# Start a local server (Python 3)
python -m http.server 8000

# Or if you have Node.js installed
npx http-server

# Then visit: http://localhost:8000
